# 🔄 Projeto Redux

A user management app built with **React + Redux Toolkit** as part of a React learning journey. Create users, manage addresses and fetch data from an external API — all with centralized state management via Redux 🗂️.

## 🚀 Features

- 👤 User login with name and email
- 🏠 Register, update and delete user address
- 📋 Fetch users list from external API
- 🔍 Fetch individual user by ID
- ⏳ Loading state feedback during async operations
- 🔀 Navigation between pages with React Router DOM
- 🧩 Async side effects handled with Redux Saga

## 🛠️ Tech Stack

- ⚛️ React
- 🗃️ Redux Toolkit
- 🔀 Redux Saga
- 🧭 React Router DOM
- 🎨 CSS Modules
- ⚡ Vite

## 📦 Installation

```bash
git clone https://github.com/your-username/projeto-redux.git
cd projeto-redux
npm install
npm run dev
```

The app will open at `http://localhost:5173` 🚀

## 🎯 How to Use

### Login:
- ✏️ Enter your **name** and **email** on the login page
- ➡️ Click **Acessar** to create your session and be redirected to the dashboard

### Managing Addresses:
- 🏠 Navigate to **Meus Endereços** to register your address
- 💾 Fill in the street and number, then click **Salvar Alteração**
- 🗑️ Click **Deletar endereço** to remove it

### Fetching Users:
- 📋 On the dashboard, click **Buscar usuários** to load all users from the API
- 🔍 Click **Buscar usuário com ID** to fetch a specific user by ID
- ⏳ A loading indicator is shown while data is being fetched
